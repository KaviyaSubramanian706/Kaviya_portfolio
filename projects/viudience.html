ViuDience: Digital Signage Retail Profiler

Timeline: 2020

Confidential Architecture Node: Original source implementation is subject to strict non-disclosure mappings. High-level infrastructural topology provided below defines open system-design evaluation workflows.

Executive Summary

An edge-deployed real-time demographic evaluation network tracking interactive digital signage usage parameters within retail environments anonymously.

Core Engineering Architecture

  • Intel OpenVINO Optimization: Engineered real-time human object detection and tracker arrays optimized specifically utilizing Intel OpenVINO pre-compiled processing runtimes.
  • Multi-Task Demographic Pipeline: Integrated parallel tracking layers resolving face tracking data metrics concurrently: Emotion, Gender matrices, Age classification, and direct Gaze alignment profiling.
  • Spatial Attentiveness Metrics: Wrote custom physical geometric scripts monitoring distance from signage clusters and target height calculations to index deep dwell time metrics.
  • Qt Application Interface: Built a standalone visualization app framework using cross-platform Qt (PyQt/C++) structures for clean deployment onto localized regional servers.
