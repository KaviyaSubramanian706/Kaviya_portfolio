📋 Executive Summary

An enterprise-grade multi-camera tracking system synchronizing a distributed Edge Master-Slave grid to mapping trajectories across large commercial boundaries like supermarkets.

🛠️ Core Engineering Architecture

  • Dual-Stream Architecture: Designed a custom YOLOv2-Tiny split network isolating independent layers for RGB and Depth streams before unified layer feature concatenation, hitting 98% localization accuracy.
  • Master-Slave Grid Topology: Distributed processing tracking algorithms on edge Jetson Nano slaves, compiling metrics to Global ID matrices over specialized Master aggregators via OC-SORT models.
  • Stepwise Feature Pruning: Implemented a proprietary gradient compression model, dropping extensive weights profiles on shallow layers to minimize footprint boundaries while securing deep logical nodes.
  • Spatial Homography Translation: Wired geometric matrices converting raw video pixel coordinates directly to physical structural 2D digital twin blueprints for tracking telemetry.
  • Immutable MLOps Lifecycle: Tied structural codebase matrices and validation targets together under single hashes using combined DVC configurations.
